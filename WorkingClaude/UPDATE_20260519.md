## 3. Cách áp dụng (5 phút)

### Bước 1: Backup
```bash
cp recommend_holistic.py recommend_holistic.py.bak_v11_orig
```

### Bước 2: Giải nén delta zip
Copy 10 files từ delta zip đè lên thư mục deploy hiện tại
(`recommend_holistic.py` sẽ ghi đè, 9 files còn lại là mới):

```bash
# Linux/macOS
unzip -o update_v11_20260519.zip -d /path/to/ba_v11/

# Windows
# Right-click zip > Extract All > chọn thư mục deploy hiện tại
```

### Bước 3: Cài thêm dependency (chỉ 1 package)

`sbv_macro_overlay.py` cần `yfinance` để pull DXY. Thêm vào requirements:

```bash
pip install yfinance
```

(Nếu environment đã có yfinance từ trước thì skip)

### Bước 4: Smoke test

```bash
python bull_div_boost.py
python sbv_macro_overlay.py
python recommend_holistic.py
```

Expected output mỗi command:
- `bull_div_boost.py`: in trạng thái BullDvg fire + max_positions (default 12)
- `sbv_macro_overlay.py`: in macro_score + regime (NORMAL/TIGHT/LOOSE) + max_pos
- `recommend_holistic.py`: chạy bình thường + 1 dòng mới `⚡ Boost overlay ACTIVE` hoặc `🛡 Defensive overlay ACTIVE` nếu có overlay fire (hôm nay 2026-05-19 chưa fire → không thấy dòng này, vẫn max_pos=12)

### Bước 5: Acknowledge SBV current state

SBV chưa có quyết định mới từ 2023-06-19. Acknowledge để silence stale warning:

```bash
python sbv_macro_overlay.py --ack-no-change
```

### Bước 6: Setup scheduled tasks (optional — chỉ Windows)

Hai task daily + hai task 12-month report:

```cmd
schtasks /create /tn BullDvgBoost /tr "%CD%\bull_div_boost_run.bat" /sc daily /st 15:15 /f
schtasks /create /tn SBVMacroOverlay /tr "%CD%\sbv_macro_run.bat" /sc daily /st 15:20 /f
schtasks /create /tn BullDvgBoostReport /tr "%CD%\bull_div_boost_report_run.bat" /sc once /sd 05/19/2027 /st 15:30 /f
schtasks /create /tn SBVMacroReport /tr "%CD%\sbv_macro_report_run.bat" /sc once /sd 05/19/2027 /st 15:35 /f
```

(Thay `%CD%` bằng absolute path nếu cần)

**Linux/macOS**: dùng cron, ví dụ:
```
15 15 * * 1-5 cd /path/to/ba_v11 && python bull_div_boost_tracker.py update >> bull_div_boost_run.log 2>&1
20 15 * * 1-5 cd /path/to/ba_v11 && python sbv_macro_tracker.py update >> sbv_macro_run.log 2>&1
```

## 4. Khi SBV announce quyết định lãi suất mới

Edit JSON hoặc dùng CLI:

```bash
python sbv_macro_overlay.py --add-event YYYY-MM-DD X.XX
```

Ví dụ nếu SBV ngày 2026-08-15 cut refi từ 4.5% xuống 4.0%:
```bash
python sbv_macro_overlay.py --add-event 2026-08-15 4.0
```

Event được sorted + dedup + ack auto-cleared. Tracker chu kỳ tiếp theo sẽ
honor event mới (nhưng cần ~90 phiên lag trước khi vào signal vì
`refi_chg_90d.shift(90)`).

Source khi có decision mới (verify):
- baochinhphu.vn / vneconomy.vn / thuvienphapluat.vn — đăng tải decision QĐ-NHNN
