# FA layer (fundamental_rating.py) research audit ‚Äî 2026-05-29

Empirical audit of the 7-axis FA layer that feeds v11/v12 stock selection (`fundamental_rating.py` ‚Üí tier A/B/C/D/E ‚Üí `recommend_holistic.py`). Methodology: Spearman IC vs forward `profit_3M` on `fundamental_rating_all.csv` (11,206 rows w/ outcome, 2014-2026), IS 2014-19 / OOS 2020+ split, regime-conditional via `vnindex_5state`, tail-risk for screens. Scripts: `fa_ic_audit.py`, `fa_ic_composites.py`, `fa_newfactor_ic.py`, `fa_sector_and_value.py`. Reports: `data/fa_ic_audit.md`, `fa_newfactor_ic.md`, `fa_sector_value.md`.

**How FA is USED (key framing):** FA tier is NOT the alpha selector in the BA book ‚Äî MEGA/MOMENTUM entries require fa_tier‚àà{C,D}+momentum+BULL; A/B route to COMPOUNDER/WAIT. FA = (a) exclusion floor (tier E=AVOID) + (b) compounder router. So "evaluate companies better" = better blow-up avoidance + better compounder ID, two different jobs.

## Findings (4 directions, all empirically tested)

1. **2 axes are dead weight ‚Äî DROP them (highest-ROI, lowest-risk).** `health` (IC ‚àí0.04 ALL, ‚àí0.09 IS) and `valuation` (‚àí0.03 ALL, ‚àí0.10 IS) have NEGATIVE IC in BOTH IS and OOS. 5-axis composite **EW5** = quality+stability+cash+shareholder+growth beats 7-axis on IC (OOS 0.089 vs 0.088, ALL 0.102 vs 0.095) AND decile spread (+6.83 vs +6.53pp OOS), both periods. CORE4 (also drop growth) best OOS IC 0.093 but smaller spread. Original designer down-weighted these (8%/10%, smallest) ‚Äî data says go further, drop. **MUST validate via integrated V11/V12 backtest before deploy** (codebase rule: never trust standalone IC).

2. **FA edge is regime-concentrated, ZERO in BULL.** total_score IC: CRISIS +0.155, EX-BULL +0.169, NEUTRAL +0.094, BEAR +0.037, **BULL ‚àí0.003**. Confirms [[feedback-bull-market-psychology]] at stock level ‚Äî fundamentals don't predict in BULL (FOMO rules). ‚Üí consider down-weighting/bypassing FA tier gating in BULL. EX-BULL: high IC but ALL tiers negative (‚àí6 to ‚àí11%) = "sell-everything" regime, FA still ranks (best falls least).

3. **Bank gap CONFIRMED + PROTOTYPE BUILT (2026-05-29).** Sector-8 IC +0.061 vs non-fin +0.124 (HALF). **Decompose by sub-code**: sector 8 = Banks 8355, RealEstate 8633 (104 tickers, lumpy), FinServices 8777, Insurance 8536 ‚Äî NOT all banks.    **INTEGRATED VALIDATION DONE (2026-05-29) ‚Äî textbook IC-vs-integrated reversal.** Swapped bank_tier‚Üígeneric for ICB-8355 in v11 BA-core classify (`fa_bank_integrated.py`, ports recommend_holistic.classify_play_type, 603k prune rows, realized profit_3M). **Standalone IC said bank model WINS (+0.047 vs ‚àí0.045 OOS) but integrated REJECTS it for BA-core**: full book annualized 70.0%‚Üí66.9% (‚àí3.1pp), bank-only OOS 93.6%‚Üí87.1%.
