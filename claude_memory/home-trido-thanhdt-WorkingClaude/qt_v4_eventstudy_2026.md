# QT v4 (Buffett) filter — per-signal event study (2026-05-29)

User reframe (correct, fixes prior mistake): stop measuring FA/quality filters as portfolio NAV (capacity/cash/Sharpe/Calmar obscure the signal). Measure the QUALITY OF THE PICKS at the stock/event level — when to buy, long-horizon outcome, when to exit. Script: `qt_v4_eventstudy.py` (reuses backtest_qt_v4 signal defs + data: qt_panel_2014_2026.pkl, fa_ratings_lh.csv, qt_v4_fin.pkl). Equal-weight per signal, unlimited capital, point-in-time fwd returns. Output: `data/qt_v4_eventstudy.md`, `data/qt_v4_events.csv`.

**QT v4 = Buffett quality-compounder filter**: universe ≥70% A/B tier (≥12Q hist), latest A/B, liq≥5B/d. Entry OR{VALUE: (pe_z<−1|pb_z<−1|dd52w<−30)&not-falling-knife; GARP: PEG∈(0,1]&NP_R≥20%&Rev_YoY≥20%&PE<25&>MA200}. Exit OR{FA_DEGRADE tier→C/D/E; OVERVALUED pe_z>2.5&pb_z>1.5; GROWTH_BROKEN 2Q NP&Rev<−15%}.

## Findings (628 entry events, 62 tickers, 2016→2026)
- **Edge is REAL but LONG-horizon** (confirms the whole session's thesis, now clean at stock level): fwd median/mean/win, excess-vs-VNINDEX: 3M +2.0%/−1.8exc/beat45% → 1Y +1.0%/−5.1exc/beat44% (LAGS index through 1Y) → 2Y +11.2%/−1.6exc (matches) → **3Y +30.4%/+52.0mean/win65%/+7.2pp exc/beat54% (BEATS index only at 3Y)**. The picks need ~3 years; portfolio-NAV sims (1-2Y, capacity-bound) structurally couldn't show this.
- **VALUE path >> GARP path**: VALUE 1Y +4.8%/2Y +15.0%; **GARP 1Y −8.9%/2Y +0.9%/beat-VNI only 24-27%** (drag). For a long-horizon quality filter, buy-cheap beats buy-growth. → drop/rework GARP.
- **Exit: OVERVALUED is GOLD, FA_DEGRADE exits too early.** OVERVALUED (n=116): median **+81.4%**, **96% win**, hold 561d — let winners run, sell when expensive (pure Buffett). FA_DEGRADE (n=217): median −2.3%/win44% (fires after damage). GROWTH_BROKEN (n=77): median −3.0%/win34% (defensive). Overall rule-exit (mean +27%, ~9mo hold) << hold-3Y (+52%) → **rules exit too early, cut compounders** (esp. FA_DEGRADE on a single-quarter tier slip).

## REINVESTMENT RUNWAY — cash-machine alone ≠ multibagger (user insight 2026-05-31, CRITICAL refinement)

Validated (BQ, asset CAGR4y / reinvest=−CF_Invest_5Y/CF_OA_5Y / ROIC5Y): **QTP = YIELD cash-cow** (power plant, asset CAGR −5%, reinvest 0.01, ROIC14% → "earnings growth" is just declining-depreciation, FINITE & fake, pays out → NO multibag); **DGC = compounder** (asset +18%, ROIC 29%); HAH(+23%/21%, cyclical), FMC(+12.5%/10%), PTB(+4.6%/13%) reinvesting; **CVT = value-DESTRUCTIVE growth** (reinvest 2.05 but ROIC 5.8% — cash machine that reinvests at poor returns); **VCS NOW matured** (asset −3.5%, reinvest 0.06 — its compounder runway was 2015-18, now a cash-return/yield profile like QTP). Key dynamics: **(a) runway is TIME-VARYING — buy compounder when runway OPEN, recognize when it closes (VCS→yield)**; (b) ROIC gates "good" vs "value-destructive" cash machines (CVT). Reclassify cash machines: COMPOUNDER{DGC,HAH,PTB,FMC} / YIELD{QTP=dividend play not multibag} / low-ROIC{CVT avoid}. Golden compounder-cheap intersection (cheap+cash-machine+runway+ROIC) = PTB, FMC (NOT QTP — yield only). To bake into screener: split CASH_MACHINE badge → COMPOUNDER (ROIC≥12% & asset-growth>0) vs YIELD (asset-growth≤0/reinvest~0).
